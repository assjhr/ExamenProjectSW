# ExamenProjectSW

# The Ultimate Webshop – Confit & Co

## Opleiding
Graduaat Programmeren  
Vak: Statisch Web  

---

## Plan van aanpak en timing

### 1. Analyse van de opdracht
- Opdracht doorgenomen en vereisten bekeken
- Inspiratie opgedaan via bestaande webshops

### 2. Structuur opzetten
- Mappenstructuur aangemaakt (frontend / admin)
- Basis HTML-structuur opgezet

### 3. Frontend (publieke webshop)
- Homepage met hero-sectie
- Productoverzicht met cards
- Productdetailpagina’s
- Responsive layout met Flexbox

### 4. Backend / Admin
- Loginpagina
- Dashboard met productentabel
- Formulier om producten toe te voegen of te bewerken

### 5. Styling
- Externe CSS-bestanden gebruikt
- CSS-variabelen in :root
- Verschillende look & feel voor frontend en admin

### 6. Testen en afwerking
- HTML en CSS nagekeken
- Navigatie getest
- Laatste aanpassingen uitgevoerd

**Timing**
- Dag 1: frontend structuur en styling
- Dag 2: productdetailpagina’s en admin login
- Dag 3: dashboard, formulier en documentatie
